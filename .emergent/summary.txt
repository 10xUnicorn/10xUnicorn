<original_problem_statement>
The user wants to build a mobile application called 10x Unicorn, designed as a mobile-first execution operating system to help individuals become high-performers.

**PRODUCT REQUIREMENTS:**
- **Core Experience:** A daily-use app centered around a Today screen.
- **Authentication:** Email + password, session persistence, password reset, account deletion.
- **Onboarding:** A quick flow to collect user's name, timezone, a 10x Goal, and a Daily Compound Habit.
- **Today Page:** The main interface, featuring:
    - Determination Slider (0-10 with emoji feedback).
    - Intention input (What kind of person are you being today?).
    - 10x Focus: The main goal for the day.
    - Five Core Actions checklist.
    - Wormhole System: A mini-CRM to log interactions with key contacts.
    - Daily Compound Habit tracker with streak counter.
    - Distraction Reflection and course-correction toggle.
    - Status Display (computed win/loss status, with manual override).
    - Date Navigation (Yesterday/Today/Tomorrow).
- **AI Course Correction Engine:** An AI chat feature to help users get back on track, using their daily inputs as context.
- **Dashboard:** A screen to show momentum metrics like streaks, completion percentages, and performance trends.
- **Wormhole Contacts:** A dedicated tab to manage contacts.
- **User Preferences:**
    - Use Emergent LLM for the AI engine.
    - Accent color: Electric purple.
    - Contact management: Allow importing from phone contacts and manual entry.
    - Notifications: Implement daily push notifications to remind users to complete their entry.
</original_problem_statement>
